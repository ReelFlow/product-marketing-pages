## How to use this index

1. Read this index whenever selecting FAQs for a page
2. Match FAQs to the page topic, problem statement, or content brief
3. Select the most relevant FAQs by CMS Name
4. Use the **Webflow Item ID** column when linking FAQs to CMS items via the Webflow API — no lookup step needed
5. The slug is provided for reference only

## Selection rules

- Pick 3 to 5 FAQs that reflect the likely next questions a searcher would ask
- Prioritise C11: Problems cluster FAQs where relevant — these align most directly with search intent
- Include at least 1 FAQ from a product-focused cluster (C1 – C9) for balance
- Maximum 2 FAQs from any single cluster
- Prefer FAQs that contain natural keyword variations and practical concerns
- Do not select generic FAQs unless they clearly support the page intent
- The FAQ section should help the page rank for adjacent long-tail searches
